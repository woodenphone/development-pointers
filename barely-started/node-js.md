# Node.JS


## Installing


### Setting up WSL + NodeJS

Open WSL:

```powershell
wsl
```


Install packages: (In WSL terminal)

```bash
sudo apt-get update && sudo apt-get upgrade -y
sudo apt install nodejs npm jq jo
```
